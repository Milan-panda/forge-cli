# Forge - Autonomous Coding Agent

Forge is a terminal-based autonomous coding agent capable of executing shell commands, file operations, and web searches.

## Installation

```bash
pip install forge
```

## Usage

```bash
forge
```

## Features

- **Autonomous Task Execution**: Interpret natural language commands.
- **Shell Integration**: Execute system commands safely.
- **File Management**: Read and write files.
- **Web Search**: Search the web and integrate results.
- **Persistent Context**: Session memory.
- **Safety**: Confirmation prompts for risky commands.
